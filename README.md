## How run tests

***
1. Install node.js https://nodejs.org/en/download/ with npm package manager

2.  Init npm

```bash
npm init -y
```
3. Install Cypress

```bash
npm install cypress
```
and

 ```bash
npx cypress -v
```
4. And start cypress test

 ```bash
npx cypress open
```

and choose Browser and start test from folder "3-test"
